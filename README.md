# Sayeed - Data Scientist Portfolio Website

A modern, professional portfolio website showcasing design services and creative work. Built with HTML5, CSS3, and vanilla JavaScript for a smooth, responsive user experience.

## Overview

This portfolio website is designed for Sayeed, a data scientist specializing in data science, python, and resources management. The site features a clean, contemporary layout with a fixed sidebar profile section and a dynamic main content area.

## Features

### Profile Section
- **Designer Profile**: Comprehensive profile card featuring name, credentials, and bio
- **Verification Badge**: Blue checkmark indicating verified professional status
- **Availability Status**: Green badge displaying "Available for work" status
- **Language Support**: Multi-language indicator (English, French, +2 more)
- **Timezone Display**: Shows designer's timezone (GMT+3)

### Skills & Tools
- Professional skills tagged and displayed (Figma, Webflow, Frontend design, +3 more)
- Visual skill representation for quick reference

### Social Media Integration
- Direct links to Instagram, Pinterest, and LinkedIn profiles
- Hover effects for enhanced interactivity

### Portfolio Showcase
- **Gallery Grid**: 6 colorful project thumbnail cards
- **Visual Diversity**: Each project displayed with distinct color schemes
- **Interactive Cards**: Clickable thumbnails for project details

### Services Section
- **Service Cards**: Multiple service offerings with descriptions
- **Pricing Display**: Clear pricing information (fixed rate and subscription options)
- **Review Ratings**: Customer review counts for social proof
- **Service Images**: High-quality visual representations of service deliverables

### Navigation
- **Sticky Top Navigation**: Main menu with Services, Products, Portfolio, and About sections
- **Active State Indicators**: Clear indication of current page/section
- **Search Functionality**: Quick search interface
- **Responsive Menu**: Mobile-friendly navigation toggle

### Call-to-Action Buttons
- **"Work with Miles"**: Outlined button for engagement
- **"Pay Miles"**: Primary black button for direct payments

## Technical Stack

## Design Specifications

### Color Palette
- **Primary Black**: `#000000` - Main text and buttons
- **White**: `#FFFFFF` - Clean backgrounds
- **Accent Green**: `#7FE157` - "Available for work" badge
- **Service Colors**: 
  - Pink: `#E89BA0`
  - Green: `#8BC34A`
  - Yellow: `#D4D946`
  - Gold: `#C9A961`

### Typography
- **Font Family**: Modern sans-serif (Inter/Segoe UI fallback)
- **Font Weights**: Regular (400), Medium (500), Bold (700)
- **Clear Hierarchy**: Distinct sizing for headings, body text, and labels

### Layout
- **Sidebar**: Fixed left panel (25% width) - Contains profile and portfolio info
- **Main Content**: Fluid main area (75% width) - Displays services and content
- **Responsive**: Mobile breakpoints for tablet and smartphone views
- **Spacing**: Consistent padding and margins throughout

## Responsive Breakpoints

- **Desktop**: Full two-column layout (sidebar + main content)
- **Tablet**: Adjustable layout with optimized spacing
- **Mobile**: Single-column layout with collapsible sidebar

## Interactions & Animations

- **Hover Effects**: Smooth transitions on buttons and service cards
- **Button States**: Visual feedback for active, hover, and click states
- **Card Animations**: Subtle scale and shadow effects on service cards
- **Social Icon Animations**: Interactive hover states on social links
- **Smooth Scrolling**: Fluid navigation between sections

## Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No build tools or dependencies required

### Installation

1. Clone or download the repository
2. Extract the files to your desired location
3. Open `index.html` in your web browser

### Usage

The website is fully self-contained:
- Simply open the HTML file in any modern browser
- Navigate using the top menu or sidebar
- Click on service cards to view details
- Use social media icons to connect with the designer
- Click call-to-action buttons to engage or make payments

## Customization

### To Update Designer Information:
- Modify the name, bio, and skills in the HTML
- Update social media links in the navigation section

### To Add Services:
- Add new service cards in the main content area
- Update pricing and review information
- Add corresponding service images

### To Change Colors:
- Update CSS variables or hex color codes in the stylesheet
- Modify the color palette to match your brand

### To Add Portfolio Items:
- Add new thumbnail cards to the portfolio grid
- Update colors and hover effects as needed
